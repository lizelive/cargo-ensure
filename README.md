# cargo sub

cargo subcrate managment

## usage

it's very powerful

### cargo sub new
```
cargo sub new [name]
```
create a subcrate, add it to the workspace

detects if subcrates are ./ or ./crates/
detects if it needs to change Cargo.toml or if it's fine

### cargo sub sync

```
add subcrates to works
```
